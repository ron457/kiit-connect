# KIIT Connect

KIIT Connect – campus navigation and info web app (HTML/CSS/JS + Firebase Auth, Firestore, Hosting).

## Features
- Search engine that tokenizes queries and matches keywords to labeled campus locations.
- Linked keywords (e.g., “canteen” → “food court”) for better recall.
- Manually labeled locations and linked keywords stored in Firestore.
- Firebase Authentication and Firestore‑backed storage for secure access.

## Tech stack
- HTML, CSS, JavaScript
- Firebase Authentication
- Cloud Firestore
- Firebase Hosting
